¿Qué es esto?

Todo website se compone de dos caras: el Back End y el Front End.
El primero se encarga de la lógica interna y el manejo de bases de datos.
El segundo representa la interfaz gráfica de un sitio web, desde la estructura hasta los estilos (colores, tipografías, texturas, secciones, etc.).
Su uso es determinante para que el usuario tenga una buena experiencia cuando navega por el sitio u aplicación.
